# TaskFlow

AI-powered task management platform built using MERN stack.

## Features

- JWT Authentication
- Create Tasks
- View Tasks
- Responsive UI
- MERN Stack

## Installation

### Backend

cd server

npm install

npm run dev

### Frontend

cd client

npm install

npm run dev

## Environment Variables

PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=supersecretkey

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Vite

Backend:
- Node.js
- Express.js
- MongoDB
